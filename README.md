# 🚑 LifeLine AI

## About the Project

LifeLine AI is an AI-powered emergency first aid web application developed for the Idea2Impact Online Hackathon 2026.

The main goal of this project is to help people during emergencies by providing quick first-aid guidance, symptom assessment, and immediate actions before professional medical help arrives.

---

## Problem Statement

During emergencies, many people do not know what to do in the first few minutes. This can delay the right treatment and increase the risk of serious injury.

LifeLine AI helps users by providing simple first-aid instructions and emergency guidance instantly.

---

## Solution

The application allows users to choose an emergency, check symptoms, understand the risk level, and view the immediate first-aid steps.

It also includes an AI Emergency Assistant that answers basic first-aid questions and helps users take quick action during emergencies.

---

## Features

- Emergency Selection
- Symptom Assessment
- Risk Level Analysis
- AI Emergency Assistant
- First Aid Guidance
- Immediate Action Recommendations
- Find Nearby Hospitals using Google Maps
- One-Tap Call to Ambulance (108)
- Emergency Helpline Information
- Responsive User Interface

---

## Supported Emergencies

- ❤️ Heart Attack
- 🔥 Burns
- 🐍 Snake Bite
- ⚡ Electric Shock
- 🩸 Heavy Bleeding
- 🫁 Choking
